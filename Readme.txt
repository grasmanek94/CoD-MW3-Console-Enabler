This custom d3d9.dll proxy adds a custom GUI with a textbox and submit button.
Place the d3d9.dll from Release folder into your CoD MW3 directory (there where your game.exe's are).
Launch Game (SP)
Use ~ key to access the console.

If you want MP Support recompile plugin, edit proxy.cpp for that.

See headers for more info.
